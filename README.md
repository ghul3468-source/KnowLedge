### 🎓 О проекте

**KnowLedge** — это десктопное приложение-справочник, разработанное для удобного доступа к информации. Программа позволяет просматривать категории, подкатегории, статьи, проходить тесты и отслеживать прогресс изучения материала.

**Тип проекта:** Итоговый проект  
**Учебное заведение:** ККОТиП  
**Группа:** ИСИП-24-01/2П

### 🚀 Установка и запуск

#### Для пользователей (EXE файл)

1. Скачайте `KnowLedge.zip`
2. Распакуйте архив
3. Запустите `KnowLedge.exe`
4. При первом запуске данные скачаются автоматически из GitHub

#### Для разработчиков

```bash
# Клонирование репозитория
git clone https://github.com/ghul3468-source/KnowLedge.git
cd KnowLedge

# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python main.py

# Сборка EXE
python build.py
